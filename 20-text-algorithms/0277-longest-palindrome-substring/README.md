# 0277 — Longest Palindromic Substring

Find the longest palindromic substring of "babad" (first of maximal length), printing `bab`. A local `expand` function captures the outer state idiomatically.

## Run

    kotlinc LongestPalindromeSubstring.kt -include-runtime -d longestpalindromesubstring.jar && java -jar longestpalindromesubstring.jar
