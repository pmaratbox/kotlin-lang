# 0083 — Sliding Window

Slide a window of size `2` over `1, 2, 3, 4` and print each window on its own line: `1 2`, `2 3`, `3 4`. `windowed(2)` returns every overlapping length-2 sublist (step defaults to 1).

## Run

    kotlinc SlidingWindow.kt -include-runtime -d window.jar && java -jar window.jar
