# 0094 — Linear Search

Linearly scan the array `4, 2, 7, 1, 9` for `7` and print the index where it is found: `found 7 at index 2`. `List.indexOf` performs a linear scan, returning the first matching index (or `-1`).

## Run

    kotlinc LinearSearch.kt -include-runtime -d linsearch.jar && java -jar linsearch.jar
