# 0062 — Do-While Loop

Use a do-while loop (whose body runs once before the condition is checked) to print `1`, `2`, and `3` on one line: `1 2 3`. `do { ... } while (cond)` runs the body once before the condition is checked.

## Run

    kotlinc DoWhile.kt -include-runtime -d dowhile.jar && java -jar dowhile.jar
