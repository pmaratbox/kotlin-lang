# 0084 — Partition

Partition `1, 2, 3, 4, 5, 6` into evens and odds, printing `evens: 2 4 6` and `odds: 1 3 5`. `partition` returns a `Pair` of two lists — matching and non-matching — destructured into `evens` and `odds`.

## Run

    kotlinc Partition.kt -include-runtime -d partition.jar && java -jar partition.jar
