# 0076 — Count Vowels

Count the vowels (`a`, `e`, `i`, `o`, `u`) in the word `hello` and print the total: `2`. `it in "aeiou"` uses the `in` (contains) operator on the string; `count { ... }` tallies the vowels.

## Run

    kotlinc CountVowels.kt -include-runtime -d vowels.jar && java -jar vowels.jar
