# 0079 — Character Codes

Print the character code of `A` and the character for code `66`: `65 B`. `Char.code` gives the character's integer code and `Int.toChar()` converts back (replacing the older `toInt()`/`toChar()` casts).

## Run

    kotlinc CharacterCodes.kt -include-runtime -d charcodes.jar && java -jar charcodes.jar
