# 0393 — Weekday After N Days

Given 2000-01-01 is a Saturday, compute the weekday 3 days later, printing `Tuesday`. Indexing an `Array<String>` by the modular sum keeps the lookup idiomatic.

## Run

    kotlinc WeekdayAfterDays.kt -include-runtime -d weekdayafterdays.jar && java -jar weekdayafterdays.jar
